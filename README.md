# 🧠 AI Text Generation App

This project is a simple Python-based AI application that generates text using a pretrained transformer model (**LaMini-Flan-T5-248M**).

---

## 🚀 About the Project

* Built using Python
* Uses a pretrained lightweight LLM
* Takes user input and generates meaningful output

---

## 📂 Important Note

Only `app.py` is uploaded in this repository.

Other files/folders like:

* model files
* virtual environment
* data folders

are **not included** due to large size.

---

## ⚙️ Setup Instructions

### 1. Clone the repository


### 2. Install Dependencies

Install the required Python libraries manually:
Dependencies Used
torch → for deep learning model execution
transformers → to load pretrained NLP model
accelerate → for efficient model loading (CPU/GPU support)
sentencepiece → for tokenizer support

```bash
pip install torch transformers accelerate sentencepiece
```

#### 📦 Dependencies Used

* **torch** → for deep learning model execution
* **transformers** → to load pretrained NLP model
* **accelerate** → for efficient model loading (CPU/GPU support)
* **sentencepiece** → for tokenizer support


### 3. Download Model

Download **LaMini-Flan-T5-248M** model manually and place it in the project folder.

---

## ▶️ Run the App



## 📝 How it Works

1. Loads pretrained model
2. Takes input from user
3. Generates response
